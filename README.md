# DashBoard-React
DashBoard with multiple widgets and services created with Raectjs.

## Features
*Drag & Drop
* Dictionary
* Change Background
* Login/Logout
* Personalise your dashboard
* Youtube Player
* Anime database
* Cryto Stocks
* News Feeder
* Weather 
* StopWatch

## Installation

    ```
    npm install
    ```
    ```
    npm run start
    ```
